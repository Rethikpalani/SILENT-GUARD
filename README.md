# SILENT-GUARD

## Description
Ce projet consiste à créer une alarme silencieuse capable de détecter une présence dans une zone.

Le but est de proposer un système simple de sécurité basé sur un capteur de mouvement.

---

## Objectif
L’objectif est de détecter une intrusion et d’envoyer une alerte.

---

## Fonctionnement
Le système fonctionne de la manière suivante :

- Un capteur PIR détecte un mouvement
- La carte UCA reçoit l’information
- Une alerte est déclenchée (LED, écran ou message)
- Un message peut être envoyé pour prévenir d’une intrusion

---

## Matériel utilisé
- Carte UCA
- Capteur PIR
- Écran OLED
- Bouton
- LED

---

## Amélioration possible
On pourrait améliorer ce système en ajoutant une communication sans fil (LoRa) pour envoyer les données sans passer par un ordinateur.
